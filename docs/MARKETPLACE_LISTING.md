# Marketplace Listing Copy

## Product Name

AI Morning OS - Premium n8n Morning Briefing Automation

## Short Description

A ready-to-import n8n workflow that sends a personalized AI-powered morning briefing to ntfy using weather data, technology news, career opportunities, and Groq AI.

## Full Description

AI Morning OS is a premium automation template for founders, operators, developers, creators, and professionals who want a useful daily briefing without building an n8n workflow from scratch.

The workflow pulls live weather from Open-Meteo, reads technology and career RSS feeds, sends the context to Groq AI, and delivers a concise morning briefing through ntfy. It includes configuration validation, retry-ready external calls, feed fallbacks, polished prompting, and complete buyer documentation.
