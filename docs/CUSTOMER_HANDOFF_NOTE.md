# Customer Handoff Note

Hello,

Your AI Morning OS n8n automation package is included in this folder.

Setup steps:

1. Import `workflow/ai-morning-os-template.json` into n8n.
2. Open **USER CONFIG - EDIT ME** and add your own details.
3. Add your Groq API key and private ntfy topic.
4. Set the preferred run time in **Schedule Trigger**.
5. Run a manual test.
6. Activate the workflow after the ntfy notification arrives successfully.

For full setup instructions, follow `docs/INSTALLATION.md`.
